# Documentação do Portal Web - Dynamics CRM
## Projeto realizado no Microsoft Dynamics 365
Toda a documentação do projeto encontra-se em Inglês ou Italiano devido a língua do Cliente.

Esse repositório trata-se da documentação da criação de um Portal Web sem banco de dados, onde os dados são consultados diretamente na API do Dynamics 365.

O cliente solicitou um portal web onde:
 - Os vendedores pudessem consultar o histórico financeiro e logístico dos clientes (pedidos, faturas, pagamentos e notas de transporte), registrar atividades de campo, como a análise técnica de pontos de venda e solicitações de materiais de marketing.
 - Os clientes pudesse consultar suas proprias Cotações / Pedidos / Faturas / Pagamentos e abrir Incidentes diretamente. 
Tudo isso sem a necessidade de ter acesso direto ao CRM.
